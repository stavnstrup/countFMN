This repository contains two XSLT stylesheets used to identify descrepenices between FMN profiles in NISP ArchiMate file created by IP CaT and the ArchiMate versions of the same FMN propfiles created by ACT.

* xPG - extract a FMN profile from a NISP ArchiMate file and convert it to a CVS file
* dumpFMN - dump an FMN profile from an ArchiMate representation of the profile

But tools extract only the minimum set of attributes necessary to identify missing or surplus standards and profiles in the NISP import.
